# Next.js 13 with Typescript reference architecture

## Getting Started

### How to run

```bash
npm install && npm dev
or
yarn && yarn dev
```

### How to build

```bash
npm run build
or
yarn build
```

### How to tests

```bash
npm test
or
yarn test
```

### More utils commands

Find and fix Javascrit and TypeScript problems according to pre-defined rules
`npm run lint`
`yarn lint`
